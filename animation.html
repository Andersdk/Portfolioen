ANIMATION

TEMAOPGAVE/SPIL

I temaet omkring animation, lærte vi at lave et interaktivt animeret spil, som vi udbyggede primært vha. Javascript. Dernæst skulle vi sætte spillet ind i en responsiv hjemmeside vi også byggede selv i HTML, CSS og lidt Javascript. Vi benyttede os af bl.a. State Machine Diagrammer, til at forstå den røde tråd i spillets funktioner. Desuden lærte vi om f.eks. Eventlisteners i Javascript, som jeg også har leget rundt med i udviklingen af min portfolio side. Desuden lærte jeg essensen af hvad Javascript variabler er og hvordan man kan bruge dem, dog var det ret svært at forstå og føler stadig jeg skal blive endnu klogere på det.
